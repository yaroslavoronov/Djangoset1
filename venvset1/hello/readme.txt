3) Создание сервиса для работы с базой данных городов, улиц, магазинов. Имеется возможность создать магазин.
4) Требуется Django с установленными библотеками:datetime, json, rest_framework
Выгрузить ветку Master с git в корень диска.
Открыть коммандную строку.
С помощью серии команд cd переместится в загруженную папку в директорию Djangoset1/venvset1/hello/.
Выполнить комманду python manage.py runserver
Перейти по появившейся ссылке.
